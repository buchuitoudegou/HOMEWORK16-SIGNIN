# 建立一个mongoDB的数据库，接口是mongodb://localhost:27017(默认接口）
* 在mongoDB/server/3.x/bin 目录下执行命令 mkdir ../db
* 在MongoDB里的server/3.x/bin/，执行命令./mongod.exe --dbpath '../db'
